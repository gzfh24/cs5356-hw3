What did I learn about the <details> HTML element?

The <details> element creates an interactive, collapsible widget. The accompanying <summary> element acts as the visible label that users can click to reveal or hide additional content to help keep the page uncluttered.

What did I learn about the mix-blend-mode CSS property?

The mix-blend-mode property controls how an element's content blends with the background. By using the difference mode, I discovered that it inverted the image colors because it calculates the difference between the image and the white background.
